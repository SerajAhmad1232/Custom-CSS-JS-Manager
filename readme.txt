Add custom CSS and JavaScript easily for all devices and even admin/dashboard area.

== Description ==
Manage custom code for different screen sizes and the admin dashboard using a built-in editor. Supports code export/import and multiple CodeMirror themes.

== Installation ==
1. Upload the plugin to `/wp-content/plugins/`.
2. Activate through the “Plugins” screen.
3. Navigate to “Custom CSS/JS” in the admin menu.

== Frequently Asked Questions ==
= Is jQuery supported? =
Yes. Wrap your code in `jQuery(document).ready()`.

= Will this code apply to all pages? =
Yes, unless scoped with conditional logic.
